# United Arab Emirates T20I Tri-Series 2025 Dataset

## Dataset Description
Comprehensive cricket statistics for the T20 tri-series between Pakistan, Afghanistan, and UAE held in August-September 2025.

## Files
- `UAE_T20_TriSeries_2025.csv` - Main dataset
- `data_dictionary.csv` - Column descriptions and metadata
- `LICENSE.md` - Usage license

## Match Coverage
- 7 complete T20 matches
- All group stage and final matches
- Team Pakistan, Afghanistan, and UAE

## Data Columns
The dataset includes:
- Match metadata (date, venue, toss, officials)
- Team performance statistics (scores, extras, boundaries)
- Player performances (top scorers, wicket-takers)
- Powerplay analysis
- Playing XIs and coaching staff

## Potential Applications
- Sports analytics research
- Cricket performance analysis
- Machine learning models
- Statistical analysis of T20 cricket
